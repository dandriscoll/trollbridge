# drawbridge

An LLM-powered HTTP/HTTPS proxy that lets LLM agents reach network
resources under controlled, inspectable, policy-governed conditions.

This repository currently contains a **design document** only. No code
has been implemented yet. The design is intended to be specific enough
that an engineering agent can implement it incrementally.

- [`DESIGN.md`](DESIGN.md) — full design document.
- [`config.example.yaml`](config.example.yaml) — example configuration.

## Status

Pre-implementation. The proxy is described, not built. See
`DESIGN.md` for the staged implementation plan.
